# Year 10 Chemistry C3 Mock Flashcards

Interactive revision flashcards based on your pasted Chemistry C3 mock questions.

## What is included

- 33 flashcards with question and answer pairs
- Core acids/alkalis, neutralisation, salts, and gas tests
- Mole calculations, concentration equations, and pH knowledge
- Energy change and reaction profile questions
- Titration method and accuracy questions
- Word and symbol equations (including balancing)

## Run locally

1. Open a terminal in this folder.
2. Start a simple server:

```bash
python3 -m http.server 8000
```

3. Open http://localhost:8000

## Use

- Click a card to flip between question and answer.
- Click Reset All Cards to flip all cards back to question side.
